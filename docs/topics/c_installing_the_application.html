How do I start using Mimo

How to install the application and set up an account

  1. Open Google Play Store and look for Mimo. Once you find it, click on the "install" button.
  2. Click on the "Get started" button in order to create a new account.
  3. Click on the "Let's go" button to continue creating your account.
  4. Choose one of the following options to state why you are interested in learning to code:
    • To become a professional developer
    • Just for fun
    • For my current job
    • For a different reason.
  5. Choose the option that decribes your occupation from the following options:
    • High school student
    • University student
    • Employee
    • None of those.
  6. Swipe to one of the following options to determine the level of your experience:
    • None
    • A little
    • A lot.
  7. Select one of the following options to state what you want to learn:
    • Web development
    • Python
    • SQL.
  8. Choose one of the following options in order to set the goal:
    • Casual (5 minutes of learning daily)
    • Regular(10 minutes of learning daily)
    • Serious(20 minutes of learning daily).
  9. Set a time for a daily reminder.
